# DesafioNorven
 Desafio PHP/WordPress para vaga de Desenvolvedor Jr

A seguir,como instalar o plugin no WordPress.
	1.	Preparação dos Arquivos do Plugin:
	•	Baixe ou clone a pasta do plugin (incluindo os arquivos principais, pastas de “includes” e “assets”) para o seu computador.
	2.	Upload dos Arquivos:
	•	Usando um cliente FTP ou o Gerenciador de Arquivos do seu serviço de hospedagem, copie a pasta do plugin para o diretório /wp-content/plugins/ do seu site WordPress.
	3.	Ativação do Plugin:
	•	Acesse o painel administrativo do WordPress, vá até Plugins > Plugins Instalados, localize o “Norven Talents Jobs” e clique em Ativar.
	4.	Configuração e Testes:
	•	Após a ativação, crie novas vagas através do menu “Vagas” no painel.
	•	Utilize o shortcode [norven_jobs] para exibir a lista de vagas em uma página do seu site.
	•	Ajuste as configurações e personalize o design conforme necessário.
